
    - 1️⃣ What is the difference between var, let, and const?
    Ans- var is the old way to declare variables in JavaScript. It can be redeclared and updated, and it does not follow block scope properly.

    let is a modern way to declare variables. It cannot be redeclared in the same scope, but it can be updated. It follows block scope.

    const is used when the value should not change. It cannot be redeclared or updated after it is declared.



    - 2️⃣ What is the spread operator (...)?
    Ans- The spread operator (...) is used to spread or copy values from an array or object. It makes it easier to combine arrays or copy data without changing the original one. For example, we can copy an array or add new values to it using the spread operator.



    - 3️⃣ What is the difference between map(), filter(), and forEach()?
    Ans- map() goes through an array and returns a new array with modified values.
    
    filter() goes through an array and returns a new array with only the elements that match a condition.

    forEach() loops through each element of an array and performs an action, but it does not return a new array.

    - 5️⃣ What are template literals?
    Ans- Template literals are a way to write strings using backticks. They allow inserting variables inside strings using ${}. They also support multi-line strings.
